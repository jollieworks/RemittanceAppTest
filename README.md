# RemittanceAppTest
Project Description: "Initialiseer een Android-project voor een MVP (Minimum Viable Product) remittance-app. De app faciliteert het versturen van geld van Nederland (EUR) naar Suriname (SRD) via een gemockte flow. De focus ligt op een modulaire Clean Architecture opzet, Jetpack Compose voor de UI, en een stabiele, moderne tech stack.
